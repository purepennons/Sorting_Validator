# Sorting_Validator
Just a tool to test a sorting list is correct or not.

# Installation
	npm install -g validate_sorting

# Usage
	sortv --inc -i <input file>

# Params
`-i`: Setting input file.

`--input`: Same as `-i`.

`--inc`(default): Sorting by samll to large.

`--dec`: Sorting by large to samll.
